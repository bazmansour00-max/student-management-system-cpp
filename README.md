 # Student Management System (C++)

A console-based student management system built using C++.  
This project allows users to manage student records efficiently.

## Features

- Add new students
- Display all students
- Search student by ID
- Delete student
- Update student information
- Sort students by:
  - GPA
  - Name
  - ID
- Display top students (based on GPA)
- Calculate average GPA
- Filter students:
  - GPA > 3
  - 2 < GPA < 3
  - GPA < 2

## Technologies Used

- C++
- Object-Oriented Programming (OOP)
- Arrays & basic data handling
